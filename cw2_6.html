• <a> for hyperlinking to relevant sites outside of the current site or directing people to other pages in the site
  • <b> for bolding letters, mostly for emphasis
  • <strong> in general it serves the same purpose as b in terms of bolding, but it helps give it more importance
  • <body> for defining all the data and graphics you actually want to show to the end user
  • <br> for breaks in text, formatting and compartementalizing data
  • <div> used to mark sections of the page, usually with different css styling
  • <h1> for titles, typically of sections, like chapter names
  • <i> to make an italic text
  • <img> for images in document
  • <ol> is an ordered list, <ul> for an unordered list
  • <li> is a list item in bulleted (ordered list)
  • <p> for paragraph
  • <span> to style part of text
